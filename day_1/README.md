# Day 1: NumPy Array Fundamentals & Creation

**Track:** NumPy

## Learning Objectives
- Create arrays via array(), arange(), linspace(), zeros/ones/full/eye
- Inspect ndim, shape, size, dtype, itemsize, nbytes
- Understand default dtypes and explicit dtype casting

## How to run
```bash
cd days/day01_array_fundamentals
python practice.py
```

## Notes
Read through `practice.py` top to bottom - each numbered section builds on
the last. Try predicting the output before running each block, then modify
the code and re-run to deepen your understanding before moving to Day 2.
