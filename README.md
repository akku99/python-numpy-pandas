# 🚀 50-Day NumPy & Pandas Practice Challenge

A structured, day-by-day practice series for **intermediate-to-expert** Python
data users, covering NumPy and Pandas in depth - from array internals and
broadcasting to groupby internals, time series, and performance tuning.
Built to be posted **one day at a time** to a GitHub repository as a public
practice log / portfolio.

## Why this repo
- 📅 **50 self-contained daily lessons** — each is a single runnable script
- 🧠 **Intermediate → expert depth** — strides, memory layout, vectorization,
  MultiIndex, chunked processing, benchmarking, not just "how to make a DataFrame"
- 🧪 **Real, runnable code** — every script executes standalone in VS Code
  (`python practice.py`) and prints its own results, no notebook required
- 🏗️ **5 capstone projects** (Days 46-50) tie everything together into
  realistic end-to-end pipelines

## Suggested daily workflow (for your GitHub streak)
1. Open `days/dayXX_topic/practice.py` in VS Code
2. Read the numbered comment sections top to bottom
3. Run it: `python days/dayXX_topic/practice.py`
4. Tweak values / break things on purpose to build intuition
5. `git add days/dayXX_topic && git commit -m "Day XX: <topic>"`
6. Push - repeat tomorrow

## Setup
```bash
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

## Requirements
See `requirements.txt`. Core: `numpy`, `pandas`. A few later days optionally
use `matplotlib` (Day 44) and `pyarrow` (Day 24, Parquet) — both are in
`requirements.txt` and each script degrades gracefully if a package is missing.
